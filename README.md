# Telegram Mini App — Финансовый помощник

## О проекте

Персональный бот для учёта доходов и расходов. Сделан для себя — чтобы быстро фиксировать траты прямо в Telegram, не переключаясь между приложениями.

## Что внутри

- **bot.py** — бэкенд на aiogram 3.x. Минималистичный: только команда `/start` и кнопка для открытия Mini App.
- **Фронтенд** — отдельный HTML-файл, который хостится на GitHub Pages. Данные хранятся в localStorage браузера.
- **База данных** — пока не подключал, всё хранится на клиенте. Если понадобится синхронизация между устройствами — допишу.
